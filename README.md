# fakeLottery

> 使用ES6构建的彩票项目

## Tech Stack

- HTML/CSS/JavaScript
- Gulp
- Node.js/Express
- Docker
- Nginx
- PM2

## Build Setup

``` bash
# 安装依赖
npm install

# serve with hot reload
npm run dev

# build for production
npm run build

# run the server
npm run server
```
